## Sistema de Backups

A partir de ahora, antes de modificar cualquier archivo, se creará un backup con el formato:
```
bk-<timestamp>_<nombre_archivo>
```

Ejemplo:
- Archivo original: `lib/screens/codes/repetition_session_screen.dart`
- Backup: `lib/screens/codes/.backups/bk-20241229_120000_repetition_session_screen.dart`

## Uso Manual
```bash
./scripts/backup_file.sh <ruta_archivo>
```

## Backups Automáticos
El asistente creará backups automáticamente antes de modificar archivos usando el script.

## Ubicación de Backups
Los backups se guardan en el directorio `.backups/` dentro del mismo directorio del archivo original.

## Límite de Backups
Se mantienen automáticamente solo los últimos 10 backups por archivo.
